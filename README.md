## Bližnjice na tipkovnici

- <i>Kopiraj</i> označeno v odložišče: Ctrl+C (**C**opy)
- <i>Izreži</i> označeno v odložišče: Ctrl+X
- <i>Prilepi</i> vsebino odložišča: Ctrl+V
